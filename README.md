# Wanderlust Travel — One Page Website

A modern, responsive one-page website for a travel company, built with pure HTML, CSS, and JavaScript.

## Live Preview

Open `index.html` in your browser to view the website.

## Features

- **Responsive Design** — Optimized for mobile, tablet, and desktop
- **Smooth Scroll Navigation** — Fixed navbar with smooth scrolling to sections
- **Animated Counters** — Stats animate when scrolled into view
- **Testimonial Slider** — Auto-play carousel with manual controls
- **Contact Form** — Form with validation and submission feedback
- **Scroll Animations** — Fade-in effects on scroll using Intersection Observer
- **Mobile Menu** — Hamburger menu for small screens

## Sections

| Section | Description |
|---------|-------------|
| Hero | Full-screen landing with stats and CTA buttons |
| About | Company introduction and key features |
| Destinations | 6 popular travel destinations with pricing |
| Services | 6 service offerings (flights, hotels, planning, etc.) |
| Testimonials | Customer reviews with slider |
| Contact | Contact info and inquiry form |
| Footer | Quick links, social media, and legal info |

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display, Noto Sans TC, Inter)

## Getting Started

```bash
# Clone the repository
git clone git@github.com:ycliao-lab/travel_company.git

# Open in browser
open index.html
```

No build tools or dependencies required.
